## Quick Start

### Try the GhostStack Demo

To get started, you can clone the repository and set it up with the following commands:

```bash
git clone <repository-url>
cd <repository>
pip install -r requirements.txt
python <main-script>.py
```

### Where's the Code?

You can find the examples in the following directory: [src/ghoststack/examples/](src/ghoststack/examples/)
