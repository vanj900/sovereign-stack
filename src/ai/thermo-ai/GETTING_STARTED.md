# Getting Started with Thermo-AI

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vanj900/Thermo-AI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Thermo-AI
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Quick Demo
To run a quick demo, execute:
```bash
python demo.py
```

## Understanding Metabolic Spine and Experiments
The metabolic spine refers to the framework through which metabolic pathways are analyzed. This project allows users to explore various metabolic experiments and their outcomes effectively. For more detailed information, refer to the documentation available in the `docs` folder.
